# pulsarsurveyscraper

## Modules:
* `pulsarsurveyscraper`

## Scripts:
* `cache_survey`: download a survey table and save to a file (HDF5).  Example: 
```
cache_survey -v -s GBNCC
```

## Requirements:
* `astropy`, `json`, `h5py`, `pyyaml`, `beautifulsoup`
